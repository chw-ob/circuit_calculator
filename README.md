# 一个简单的电路可视化计算软件
本软件计划以Python的tkinter图形库为基础，
## 注意事项
组员提交的branch以自己名字的中文拼音手写命名，比如陈汉文提交的branch就应该是chw，尽量不要使用全局变量.需要的变量以函数变量作为输入即可。
## 规划的功能
1. 创建一个元件库（包含直流和正弦电压源,电阻,电感,电容），允许用户借助元件库自定义电路
2. 计算任意两节点间的等效电阻
3. 计算电路在直流和正弦激励下的响应
